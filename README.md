# My Website

This is a simple website hosted using GitHub Pages.

## Features
- Responsive design
- Interactive button
- Clean UI

## How to Run
1. Open `index.html` in browser

## GitHub Pages Deployment
1. Go to repository
2. Click **Settings**
3. Go to **Pages**
4. Select branch: `main`
5. Save

Your site will be live at:
https://your-username.github.io/repository-name/
